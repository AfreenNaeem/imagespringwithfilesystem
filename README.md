# Image Upload Management System

A backend project developed using Java, Spring Boot, REST APIs, and MySQL for managing image upload operations.

## Features

* Upload Images
* View Images
* Update Images
* Delete Images
* File System Storage
* REST API Integration

## Technologies Used

* Java
* Spring Boot
* REST APIs
* MySQL
* Maven
* Postman

## Project Purpose

This project was developed to learn backend development, file handling, REST APIs, and database integration using Spring Boot.
